Pydoop Documentation¶

Welcome to the documentation for Pydoop 1.0.0-rc2. Pydoop is a package that provides a Python API for Hadoop. To get started, read the tutorial.

How to Cite¶

Pydoop is developed and maintained by researchers at CRS4 – Distributed Computing group. If you use Pydoop as part of your research work, please cite the HPDC 2010 paper.

Plain text:

S. Leo and G. Zanetti.  Pydoop: a Python MapReduce and HDFS API for
Hadoop.  In Proceedings of the 19th ACM International Symposium on
High Performance Distributed Computing, 819-825, 2010.

BibTeX:

@inproceedings{Leo:2010:PPM:1851476.1851594,
